# Guía pedagógica ambiental · Red Escolar de Pluviómetros de Medellín

## Propósito
Convertir cada sede educativa en una estación escolar de observación climática, integrando medición de precipitación, análisis de datos, gestión del agua y acciones de adaptación ambiental.

## Componentes
1. Medición diaria con protocolo común.
2. Dashboard territorial por comuna, corregimiento, barrio, vereda y núcleo.
3. Aula Climática con actividades por grado.
4. Misiones Guardianes Climáticos.
5. Semáforo de calidad del dato.
6. Calculadora de aprovechamiento de agua lluvia.
7. Ficha ambiental por sede.

## Protocolo mínimo
- Leer el pluviómetro a la misma hora.
- Registrar 0 mm cuando no llueva.
- Vaciar el instrumento después de cada lectura.
- Verificar limpieza, nivelación y ubicación.
- Registrar eventos relevantes como tormenta, granizo o inundación cercana.

## Uso pedagógico
La red puede articular Ciencias Naturales, Matemáticas, Tecnología, Geografía, PRAE, investigación escolar y cultura ciudadana ambiental.
